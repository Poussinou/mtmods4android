Privacy Policy

We Collect

  • Whenever the app connects to our server, for example to download the available mods list or to report a mod1.
    • IP Addresses
    • Request headers, such as the device user-agent (OS, processor, device, etc, if available)
  • When you install a mod from the available mod list
    • The mod's name and author
    • Download url
    • Download size
    • The download's response code
    • Vague error messages such as "wrong-response", "wrong-content-type".
  • When reporting a mod
    • Only the information seen in the report form.

Reasons for collection

We collect this infomation in order to improve our available mods list. Reporting on downloads allows us to detect popularity. Reporting mods allows rogue mods to be removed.

Length of Storage

We may store this information permanently.
Logs (1) will probably be cleared occasionally.
